# 🌿 AI Plant Doctor

AI-powered plant disease detection system using TensorFlow and EfficientNet.

Upload a leaf image to get:
- Disease prediction
- Confidence score
- Remedy suggestions
- Live weather insights
- Smart watering advice
- Product recommendations

---

## 🚀 Tech Stack
- Python
- TensorFlow / Keras
- EfficientNet
- Gradio
- NumPy
- Pillow
- wttr.in Weather API

---

## 📸 How It Works
1. Upload plant leaf image.
2. Model predicts disease.
3. System generates medical-style report.
4. Weather + watering advice included.
